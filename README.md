## 프로젝트 소개
일상 공유 및 소통하는 블로그

## 개발 환경
- Node.js + Express + React
- MariaDB 10.3.34
- Sass

## 서버
- AWS EC2 Ubuntu(20.04.1 LTS)
- AWS S3 
- Nginx 

## 주요 기능
 **로그인**
 - DB값 검증
 - 카카오톡 로그인 API
 - 로그인 시 accesstoken, refreshtoken 쿠키 및 로컬 생성
 
**회원가입**
 - DB값 검증
 - 카카오톡 로그인 API

**글작성**
 - 작성, 읽기, 수정, 삭제(CRUD)
 
 **댓글 & 대댓글**
 - 작성, 읽기, 수정, 삭제(CRUD)
